Welcome to BUGSER!

This is the newest issue tracker app made for you.

As part of the FSDI course this project presents an issue tracker web application, where you are able to create a user with one of three roles (customer, agent and manager).
Issues are made up of a title, summary, description, requester, assignee and one of three status (To Do, In Progress and Done).
You are able to create issues to track with any account role but only agents and managers are allowed to be selected as assignee.

In the "Issues" section you are provided with a view of all issues you have created on the site, if you are a manager then you can view all issues that exist.
You also have access to a navbar to view issues filtered by their status.
